# Chiru SQL

🚀 A lightweight SQL toolkit for Node.js.

Chiru SQL provides a clean, simple API for working with MySQL and PostgreSQL without the complexity of traditional ORMs.

## Features

- ⚡ Lightweight
- 🚀 Fast
- 📦 Zero decorators
- 🛠 Simple CRUD
- 🔍 Query Builder
- 🔄 Transactions
- 📝 Raw SQL Support
- 🔐 Parameterized Queries
- 🌐 MySQL & PostgreSQL Support
- 💙 TypeScript Ready
